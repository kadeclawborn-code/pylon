# Pylon LLC

**A synthetic, open-source B2B SaaS company built as a benchmark substrate for AI agent tooling.**

If you're building or evaluating AI agents that operate inside an enterprise — code review, ticket triage, support automation, sales coaching, BI assistants, security scanners — you've probably written stub fixtures. Pylon is meant to replace those stubs with a coherent, multi-domain fake company that AI tools can be benchmarked against.

Pylon is fictional. Every customer, employee, codebase, and event is synthetic. The plausibility is the point: it should look real enough that an LLM reasons about it as a coherent enterprise, while being entirely controllable for testing.

## Status

🚧 **Early scaffolding.** Tier 0 (the minimum viable substrate) is being built as a series of public POCs by [Bent Wire Studio](https://github.com/kadeclawborn-code/bentwire). Watch for the first releases under `v0.1.0`.
